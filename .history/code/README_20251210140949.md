# code 目录说明

本目录包含本次实验的示例代码与运行说明。文件列表与用途：

- `hello_python.py`：简单的 Python 示例，打印问候并计算两数之和，支持命令行参数或交互式输入。
- `sum_cpp.cpp`：简单的 C++ 程序，支持通过命令行参数或标准输入输入两个整数并输出和。
- `hello_ros.py`：ROS（Noetic）Python 节点示例，发布字符串消息到话题 `hello_topic`。
- `turtle_circle.py`：ROS（Noetic）Python 节点示例，通过发布 `/turtle1/cmd_vel` 控制 turtlesim 画圆。
- `multi_turtle.launch`：ROS launch 文件，启动两个命名空间下的 `turtlesim_node` 实例用于多乌龟演示。

运行与使用说明（在 Ubuntu + ROS Noetic 环境下）：

1. Python 示例：

```bash
# 直接运行（需要 Python3）
python3 hello_python.py

# 传入两个参数示例
python3 hello_python.py 4 5
```

. C++ 示例：

```bash
# 在安装了 g++ 的 Linux/Windows 环境下：
g++ sum_cpp.cpp -o sum_cpp
./sum_cpp       # 交互式输入
# 或指定参数
./sum_cpp 3 7
```

ROS 示例（仅在已安装 ROS Noetic 的 Ubuntu 20.04 上运行）：

```bash
# 确保已 source ROS 环境（例如：source /opt/ros/noetic/setup.bash）
# 给脚本可执行权限（仅需一次）
chmod +x hello_ros.py turtle_circle.py

# 在一个终端启动 roscore
roscore

# 另一个终端运行 turtlesim
rosrun turtlesim turtlesim_node

# 运行画圆节点
rosrun <your_package> turtle_circle.py

# 运行 hello ROS 节点（示例）
rosrun <your_package> hello_ros.py

# 使用 launch 启动两个 turtlesim（需要把 launch 放在 catkin 包的 launch/ 下）
roslaunch <your_package> multi_turtle.launch
```
